# SEO Cheatsheet (2025-2026)

Quick reference for the most important SEO optimizations.

---

## Title Tags
```
✅ Under 60 characters
✅ Primary keyword at start
✅ Unique per page
✅ Compelling (drives clicks)
```

**Formula:** `[Primary Keyword]: [Benefit] | [Brand]`

---

## Meta Descriptions
```
✅ 150-160 characters
✅ Include primary keyword
✅ End with CTA
✅ Match search intent
```

**Formula:** `[Hook]. [Key benefit]. [CTA].`

---

## Headers
```
✅ One H1 per page (main keyword)
✅ H2s for major sections
✅ H3s for subsections
✅ Don't skip levels
✅ Include keyword variations
```

---

## Core Web Vitals

| Metric | Target | Measures |
|--------|--------|----------|
| **LCP** | ≤ 2.5s | Loading |
| **INP** | ≤ 200ms | Responsiveness |
| **CLS** | < 0.1 | Visual stability |

---

## AI SEO Quick Wins

### Content Structure
- [ ] Add FAQ section (4-8 questions)
- [ ] Add TL;DR summary
- [ ] Use short paragraphs (2-3 sentences)
- [ ] Include statistics with sources
- [ ] Make statements quotable

### Schema Markup (JSON-LD)
Priority types:
1. `FAQPage`
2. `HowTo`
3. `Article`
4. `Person` (author)
5. `Organization`

---

## Internal Linking
```
✅ 3-10 contextual links per page
✅ Descriptive anchor text
✅ Link to pillar pages
✅ Fix orphan pages
✅ No "click here" anchors
```

## Quick Audit Checklist

### Technical
- [ ] HTTPS enabled

- [ ] Core Web Vitals passing

- [ ] Sitemap submitted
- [ ] No broken links

### On-Page
- [ ] Title tag optimized
- [ ] Meta description present
- [ ] H1 with keyword

- [ ] Internal links present

### AI SEO
- [ ] Schema markup added
- [ ] FAQ section present

- [ ] Statistics included
