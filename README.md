# SEO Master - Claude Code Skill

> A comprehensive Claude Code skill for SEO audits, content optimization, and AI SEO (GEO) strategy.

**Works with both traditional Google SEO and AI search engines (ChatGPT, Perplexity, Gemini, Google AI Overviews).**

---

## What This Does

This Claude Code skill helps you:

- **Audit** pages for SEO issues (traditional + AI)
- **Optimize** content for search engines and LLMs
- **Generate** comprehensive SEO strategies
- **Implement** best practices for 2025-2026

## Quick Start

### Installation

1. Clone this repo into your project or home directory:
```bash
git clone https://github.com/wikatar/Skills-Claude-SEO.git ~/.claude/skills/seo-master
```

2. Or copy the skill file directly:
```bash
mkdir -p ~/.claude/skills
cp .claude/skills/seo-master.md ~/.claude/skills/
```

3. The skill will be available in Claude Code automatically.

### Usage Examples

```
# Audit a page
"Audit this page for SEO: https://example.com/blog-post"

# Optimize content
"Optimize this content for the keyword 'best CRM software for startups'"

# Generate strategy
"Create an SEO strategy for my e-commerce site selling outdoor gear"

# AI SEO specific
"How should I structure my FAQ page for AI search engines?"
```
